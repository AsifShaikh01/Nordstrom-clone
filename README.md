# NordStrom Clone
**This is a solo project**


A clone of an e-commerce website by  using React and chakra UI.An E-commerce website project developed in 6 days where user can singup, login with an email. User can see different pages and different products which are categorised well. Going forward user can add product to cart and can handle cart easily and finally can checkout.
